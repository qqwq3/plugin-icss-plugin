# 年份范围选择框

利用两个 DatePicker 组件，来实现年份选择功能。使用 `v-model` 实现双向绑定，数组内只支持 `Date` 格式。

支持如下属性：

- size 控制输入框大小，同 element 的 DatePicker 组件。支持 `large`, `small`, `mini`
- clearable 是否显示清除按钮
- editable 文本框是否可输入

**注意，不支持其他 DatePicker 组件属性和方法！**
